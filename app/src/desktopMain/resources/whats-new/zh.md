# [2.1.6] - 2026-07-07

## 🔁 修复 Chrome 扩展与桌面端的剪贴板死循环
修复了 Chrome 扩展与桌面端同时运行时,同一条剪贴板内容可能被两端不停地来回同步、历史记录被重复条目刷屏的问题。现在两端能正确识别对方写入的内容,不再互相触发。

## 🪟 Windows 剪贴板监听升级
Windows 上的剪贴板监听切换到了系统推荐的事件通知机制,复制的内容能被更及时、更可靠地捕获。同时修复了来源识别问题——历史记录中显示的"来自哪个应用"现在取自真正写入剪贴板的程序,而不是当时恰好在前台的窗口。

## 🪟 适配移除了 wmic 的新版 Windows
较新的 Windows 11 已移除 wmic 工具,导致 CrossPaste 无法生成设备标识。现在改从系统注册表读取,在新版 Windows 上也能正常工作。

## 🔍 搜索窗口不再"记仇"
重新打开搜索窗口或修改搜索词后,列表现在会回到顶部并选中最新的条目,不再停留在上次浏览的位置或选中一条已经对不上号的内容。

## 📶 设备发现优先走局域网
当电脑同时有多个网络连接(例如蜂窝网卡或手机共享网络)时,CrossPaste 现在会优先使用局域网来发现附近设备,发现和连接更可靠。

## 🐧 新增 Linux ARM64 版本
CrossPaste 现在提供 Linux ARM64(aarch64)安装包,ARM 架构的 Linux 设备也能用上了。

# [2.1.5] - 2026-06-11

## 🛠️ 修复"检查更新"报错
修复了在有新版本可用时,点击"检查更新"会弹出错误(Cannot call invokeAndWait from the event dispatcher thread)的问题。该问题影响 Windows 安装版与 macOS 用户,现在手动检查更新可以正常工作了。

# [2.1.4] - 2026-06-10

## 🔄 设备连接更稳、重连更快
这一版我们重点打磨了设备之间的连接。当你切换 Wi-Fi、网络地址发生变化、或电脑从睡眠中唤醒后,CrossPaste 现在能更快地重新发现彼此并恢复连接,不再长时间卡在"离线"状态。即使设备的 IP 变了,也能可靠地重新连上。

## 🆕 应用内更新日志与升级提醒
现在你可以直接在应用内查看每次更新带来了哪些新变化。有新版本可用时,左侧菜单会出现一个小红点提醒你升级。

## ⬆️ Windows(免安装 zip):应用内一键更新
Windows 免安装(zip)版现在可以在应用内完成更新。有新版本时,CrossPaste 会自动下载并校验,随后替换为新版本并重启——不必再手动下载、解压新的 zip。

## 🖱️ 修复粘贴循环问题
修复了在某些情况下按 Cmd/Ctrl+V 可能触发无限粘贴的问题,粘贴行为现在更可靠。

## ✨ 更顺滑的设备列表
重新检测离线设备时,设备列表不再闪烁,显示更稳定。

## 🔤 修复富文本复制乱码
修复了从 JetBrains 系列 IDE 等应用复制带格式文本时出现乱码的问题。剪贴板内容的编码识别现在更加可靠。

## 🐧 Linux 剪贴板体验改进
在 Wayland 桌面环境下,复制的内容现在能被更快地捕获;在 Hyprland 和 Sway 上,CrossPaste 还能正确识别内容是从哪个应用复制的。

# [2.1.3] - 2026-05-25

## 📤 跨设备传文件更快了
设备之间传文件现在更快、更稳。复制文件的那台设备会直接把内容送到你的其他设备，不用再等对方来"要"数据。

## 🅾️ 更清爽的 OCR 语言设置
图片取字（OCR）的语言设置界面焕然一新，每种已加载的语言都有独立卡片。你还可以上下拖动调整优先级，决定遇到多语言图片时先尝试哪种语言。

## 🛡️ Windows：网络被拦截时会主动提醒
在 Windows 上，如果当前网络被设为"公用"，系统防火墙会悄悄挡住 CrossPaste 发现附近设备。现在 CrossPaste 会自动检测这种情况，并提供一键跳转到对应的 Windows 设置，方便你把网络切换为"专用"。

## 🖱️ 拖拽更顺手
从 CrossPaste 侧边栏把内容拖到其他应用的体验大幅改善。在 macOS 上，拖拽时的预览不会再被粘贴窗口挡住——你终于能看清自己在拖什么了。即使屏幕上有大量条目，拖拽依然流畅。另外，搜索标签现在也支持拖动排序了。

## 🖼️ 侧边预览支持动图与真实视频缩略图
剪贴板历史里的 GIF 现在会真正动起来（即使是大图也很流畅），视频文件会显示首帧的真实缩略图，而不再是一个通用图标。

# [2.0.0] - 2026-04-26

## 🌍 Chrome 扩展正式成为一等公民
CrossPaste 现在提供 Chrome 扩展，作为真正的平台加入同步网络。它支持图片预览、大文件下载、右键复制，并在桌面端同机运行时自动让位。

## 🔌 全新 WebSocket 同步协议
全双工的 WebSocket 同步协议取代了原先的轮询方式，配合增量拉取和主动存活探测，断线后能更快恢复连接。

## 🔐 更安全的配对方式
新的短验证码（SAS）配对协议可在设备配对过程中防止中间人攻击，让建立信任更安全、更省心。

## 🇵🇹 新增葡萄牙语
全应用新增完整的葡萄牙语翻译。
